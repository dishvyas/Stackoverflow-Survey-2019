Using data from Stackoverflow 2019 survey answered the following queries:
a. Find the average age of developers when they wrote their first line of code.
b. Deduce the percentage of developers who know python in each country.
c. Generate a report for the average salary of developer based on continent.
d. Based on this survey, what will be the most desired programming language for the
year 2020?
e. What is the distribution of people who code as a hobby based on gender and
continent?
f. Generate the report for job and career satisfaction of developer based on their
gender and continent?
g. For problems f and g, gender should have three categories man, woman and others.
If a record have any other value, consider it as Others.



Unzip the developer survey zip file at the root of this directory.

Run the python files to see the results being displayed according to the queries given above in Python3 by using the following command :
	python3 nameofthefile.py
